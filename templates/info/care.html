Догляд за шкіряними виробами

Загальні рекомендації

Шкіряні вироби потребують особливого догляду для збереження зовнішнього вигляду та продовження терміну служби. Дотримуючись наших порад, ви зможете насолоджуватись своїми виробами багато років.

Уникайте вологи

Натуральна шкіра чутлива до вологи. Уникайте потрапляння води на виріб. Якщо це сталося, обережно промокніть вологу сухою м’якою тканиною та дайте виробу висохнути природним шляхом, подалі від джерел тепла.

Захищайте від сонця

Тривалий вплив прямих сонячних променів може призвести до вигоряння та пересихання шкіри. Зберігайте вироби подалі від вікон і не залишайте їх під сонцем.

Уникайте екстремальних температур

Не зберігайте шкіряні вироби біля опалювальних приладів або в надто холодних приміщеннях. Оптимальна температура зберігання: 15–25°C.

Використовуйте спеціальні засоби

Для догляду за шкірою застосовуйте лише спеціальні засоби, що підходять до типу вашої шкіри. Не використовуйте побутові мийні засоби, спирт або розчинники.

Регулярний догляд

Очищення

  1. Видаліть пил та бруд м’якою сухою тканиною або щіткою з м’якою щетиною
  2. Для ретельнішого очищення використовуйте спеціальний засіб для шкіри
  3. Нанесіть невелику кількість засобу на м’яку тканину й обережно протріть поверхню
  4. Дайте виробу повністю висохнути

Живлення та зволоження

  1. Після очищення нанесіть спеціальний крем або бальзам для шкіри
  2. Рівномірно розподіліть засіб тонким шаром по всій поверхні
  3. Дайте засобу вбратися протягом кількох годин
  4. За потреби відполіруйте виріб м’якою тканиною

Регулярність догляду

  • Очищення від пилу: щотижнево
  • Ретельне очищення: раз на місяць або за потреби
  • Живлення і зволоження: раз на 2–3 місяці

Догляд за різними типами шкіри

Гладка шкіра

Гладка шкіра потребує регулярного догляду для збереження блиску та запобігання тріщинам.

  • Очищуйте м’якою тканиною або спеціальною щіткою
  • Використовуйте крем, що відповідає кольору виробу
  • Після нанесення крему відполіруйте м’якою тканиною

Замша та нубук

Замша та нубук потребують особливого догляду через оксамитову текстуру.

  • Використовуйте спеціальну щітку для замші
  • Не використовуйте креми або рідкі засоби
  • Застосовуйте спеціальні спреї для замші та нубуку
  • Для видалення плям використовуйте гумку для замші

Шкіра рослинного дублення

Шкіра рослинного дублення з часом набуває гарної патини.

  • Регулярно очищуйте від пилу м’якою тканиною
  • Використовуйте натуральні олії й воски для живлення
  • Не застосовуйте засоби на водній основі

Зберігання шкіряних виробів

Правила зберігання

  • Зберігайте вироби у сухому, добре провітрюваному місці
  • Використовуйте чохли з бавовняної тканини для захисту від пилу
  • Не зберігайте вироби в поліетиленових пакетах
  • Сумки зберігайте, наповнивши їх папером для збереження форми
  • Пояси краще зберігати скрученими або на спеціальних вішалках
  • Гаманці зберігайте в розправленому вигляді

Довготривале зберігання

Перед тривалим зберіганням:

  1. Ретельно очистіть виріб
  2. Обробіть живильним кремом або олією
  3. Дайте засобу повністю вбратися
  4. Покладіть виріб у бавовняний чохол
  5. Зберігайте в сухому, темному та добре вентильованому місці
